# Prompt สำหรับนำไปสั่งงาน AI ในเครื่องหลัก (Main Server)

ให้คัดลอกข้อความด้านล่างนี้ไปวางให้ AI ในเครื่องหลักที่มีการตั้งค่า Database และ Server เตรียมไว้แล้ว เพื่อให้ AI สามารถทำงานต่อจากเครื่องนี้ได้ทันทีครับ

---

"ตอนนี้ฉันนำโค้ดที่อัปเดตหน้า UI ล่าสุด (ลบระบบสัมมนาเก่าทิ้ง, ทำหน้า Chatbot Dashboard สไตล์ E-commerce, และเพิ่มปุ่มตัวกรองแชท) มาลงในเครื่องนี้แล้ว 

เครื่องนี้มีการตั้งค่า Database และ Server เตรียมไว้แล้ว ฉันต้องการให้คุณช่วยทำสเต็ปต่อไปนี้ให้หน่อย:
1. **ตรวจสอบ Database:** ตรวจสอบไฟล์ `.env` ว่าค่า `DATABASE_URL` ถูกต้องไหม จากนั้นรันคำสั่ง `npx prisma db push` หรือ `migrate` เพื่อสร้างตารางให้ตรงกับ Schema ล่าสุด (User, ChatSession, ChatLog, Admin, AiAgent)
2. **รันทดสอบ Server:** ช่วยรัน Backend Middleware (ที่ทำหน้าที่รับ Webhook) และรัน Frontend (Next.js) ให้ทำงานพร้อมกัน เพื่อเช็คว่าเชื่อมต่อ Database ผ่านไหม
3. **เชื่อม API กราฟ:** ช่วยแนะนำหรือเริ่มเขียนโค้ดในการดึงข้อมูลแชทจริงๆ จากฐานข้อมูล มาผูกเข้ากับ Component กราฟในหน้า `/chatbot-dashboard` ที่เราทำ UI รอไว้ให้หน่อย
4. **ดึงรูป Profile จริง:** ช่วยอัปเดต API `fetchChatData` ให้ดึงรูปโปรไฟล์ลูกค้าของจริงจาก Facebook/LINE มาโชว์แทนรูปจำลอง (Mock) ของเทมเพลต"
