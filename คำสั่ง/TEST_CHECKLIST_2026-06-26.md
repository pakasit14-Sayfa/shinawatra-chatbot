# เช็คลิสต์ทดสอบ — สิ่งที่แก้ไป (2026-06-26)

ทดสอบโดยทักแชทจริงเข้าเพจ (หรือยิง webhook ตรงก็ได้) ทำตามลำดับขั้นในแต่ละเคส
ใช้เบอร์/แอคเคาท์ทดสอบ **ไม่ใช่ลูกค้าจริง** เพื่อกัน data ปนกับของจริง

> หมายเหตุ: เพจ 1 = Brain 1 (พิษณุโลก) / เพจ 2 ทั่วไป+เพจ 3 พยาบาล = Brain 2 (ชินวัตร)

---

## 1. Brain 1 (พิษณุโลก) — แก้ multi-program false positive

### 1.1 เคสที่ต้อง "ไม่" ถามแบบกำกวม (เคสนี้คือบั๊กที่แก้)
พิมพ์ (ข้อความเดียว):
```
สนใจป.เอกรัฐประศาสนศาสตร์ค่ะ
```
**ผลที่ควรได้:** ระบบเข้าใจว่าหมายถึง ป.เอก หลักสูตรเดียว แล้วถามข้อมูลที่ขาด (วุฒิ/อาชีพ) ต่อ — **ห้าม**ถามแบบ "สนใจหลายหลักสูตรเลยนะคะ เลือกก่อนได้ไหมคะ..."

- [ ] ผ่าน

### 1.2 เคสที่ต้อง "ยัง" ถามแบบกำกวม (กันไม่ให้แก้บั๊กเกินจำเป็น)
พิมพ์:
```
สนใจป.โทกับป.เอกค่ะ
```
**ผลที่ควรได้:** ระบบถามให้เลือกก่อนว่าสนใจระดับไหน (เพราะพูดถึง 2 หลักสูตรจริง)

- [ ] ผ่าน

## 2. Brain 2 (ชินวัตร) — PN ไม่บังคับถามเกรดอีกแล้ว

ทักเข้า **เพจ 3 (พยาบาล)** พิมพ์ทีละข้อความ:
```
1. สนใจผู้ช่วยพยาบาลค่ะ
2. จบ ม.6 ค่ะ
3. อายุ 20 น้ำหนัก 50 ส่วนสูง 160
```
**ผลที่ควรได้:** หลังข้อความที่ 3 ต้องได้รายละเอียดคอร์ส + รูปตารางค่าเทอมทันที **ห้าม**ถูกถามเรื่องเกรดเลยตลอดทั้งบทสนทนา

- [ ] ผ่าน

(เทียบกับ RN ที่ยังต้องถามเกรดตามปกติ — ลองพิมพ์ "สนใจพยาบาลศาสตร์ค่ะ" ที่เพจ 3 แทน ข้อความที่ 1 จะเห็นว่ามีถามเกรดด้วย ถือว่าถูกต้อง)

## 3. Brain 2 — รูปตารางค่าเทอม HR ครบ 9 เคส

ทักเข้า **เพจ 2 (ทั่วไป)** ทดสอบทีละ combo ด้านล่าง (เริ่มบทสนทนาใหม่ทุกครั้ง/เปลี่ยนคนคุยถ้าจะเทสหลายเคส):

| ลำดับข้อความ | ระยะเวลาที่ควรได้ | รูปที่ควรแนบ |
|---|---|---|
| สนใจ HR ค่ะ → จบปวส.บัญชีค่ะ → ทำงานเอกชนค่ะ | 1.5 ปี | hr-1.5y-private |
| สนใจ HR ค่ะ → จบปวส.สังคมค่ะ → ทำงานเอกชนค่ะ | 2 ปี | hr-2y-private |
| สนใจ HR ค่ะ → จบป.ตรีวิทย์ค่ะ → ทำงานรัฐวิสาหกิจค่ะ | 2.5 ปี | hr-2.5y-gov |
| สนใจการจัดการค่ะ → จบม.6ค่ะ → ไม่มีประสบการณ์ค่ะ → ทำงานเอกชนค่ะ | 3.5 ปี | hr-3.5y-private |
| สนใจการจัดการค่ะ → จบม.6ค่ะ → ไม่มีประสบการณ์ค่ะ → ทำงานรัฐวิสาหกิจค่ะ | **3 ปี** (ไม่ใช่ 3.5!) | hr-3y-gov |
| สนใจการจัดการค่ะ → จบปวช.ค่ะ → ประสบการณ์ 5 ปีค่ะ → ทำงานเอกชนค่ะ | 3 ปี | hr-3y-private |
| สนใจการจัดการค่ะ → จบปวช.ค่ะ → ประสบการณ์ 15 ปีค่ะ → ทำงานรัฐวิสาหกิจค่ะ | 2.5 ปี | hr-2.5y-gov |

**ทุกเคสต้องมีรูปแนบ ไม่มีเคสไหนที่ตอบแบบไม่มีรูปเลย**

## 4. Brain 2 — รูปหลักสูตร ป.โท บริหารการศึกษา

ทักเข้า **เพจ 2 (ทั่วไป)**:
```
1. สนใจป.โทบริหารการศึกษาค่ะ
2. มีแล้วครับ
```
**ผลที่ควรได้:** สรุปรายละเอียดหลักสูตร + แนบรูปโครงสร้างหลักสูตร/การรับรองคุรุสภา

- [ ] ผ่าน

ทดสอบกรณีไม่มีใบประกอบวิชาชีพครู:
```
1. สนใจป.โทบริหารการศึกษาค่ะ
2. ยังครับ
```
**ผลที่ควรได้:** ขออภัย ไม่สามารถเรียนได้ (เพราะจำเป็นต้องมีใบประกอบวิชาชีพครู)

- [ ] ผ่าน

## 5. Brain 2 — คำพูดลำลองที่คนไทยใช้จริง (เพิ่งแก้)

### 5.1 HR — เรียก "ฝ่ายบุคคล" แทนชื่อทางการ
ทักเข้า **เพจ 2 (ทั่วไป)**:
```
สนใจฝ่ายบุคคลค่ะ
```
**ผลที่ควรได้:** ต้องเข้าใจว่าหมายถึงสาขา HR แล้วถามวุฒิต่อ (ไม่ใช่เด้งเมนูทั่วไป)

- [ ] ผ่าน

### 5.2 Engineering — "ปริญญาคู่" แทน "2 ปริญญา"
```
สนใจวิศวกรรมปริญญาคู่ค่ะ
```
**ผลที่ควรได้:** ต้องเข้าใจว่าเลือกหลักสูตร 2 ปริญญา (เครื่องกล+ความปลอดภัย) แล้วถามวุฒิต่อ

- [ ] ผ่าน

### 5.3 ป.บัณฑิต — "ครูอัตราจ้าง" แทน "เป็นครู"
```
1. สนใจป.บัณฑิตวิชาชีพครูค่ะ
2. เป็นครูอัตราจ้างค่ะ
```
**ผลที่ควรได้:** ข้อความที่ 2 ต้องถูกตีความว่า "เป็นครู" แล้วได้รายละเอียดคอร์สเลย (ไม่ถูกถามซ้ำ)

- [ ] ผ่าน

### 5.4 กันไม่ให้แก้ข้อ 5.3 เกินจำเป็น (regression) — "พี่เลี้ยง" ต้องยังถูกปฏิเสธ
```
1. สนใจป.บัณฑิตวิชาชีพครูค่ะ
2. ไม่ได้เป็นครูผู้สอน เป็นพี่เลี้ยงในสถานศึกษาค่ะ
```
**ผลที่ควรได้:** ขออภัย ไม่สามารถเรียนได้ (เพราะไม่ได้เป็นครูผู้สอนจริง)

- [ ] ผ่าน

## 6. Google Sheets — เก็บคำถามที่ AI ตอบไม่ได้

ทักถามคำถามที่ระบบไม่มีข้อมูลตอบ (จะถูก deflect ไปแอดมิน) เช่น:
```
มีหอพักให้นักศึกษาไหมคะ
```
**ผลที่ควรได้:**
1. บอทตอบ "เดี๋ยวอาจารย์ติดต่อไปให้ข้อมูลเพิ่มเติมนะคะ" (หรือคล้ายกัน)
2. เปิด Google Sheet **"Brain - คำถามที่ AI ตอบไม่ได้"** ต้องมีแถวใหม่ขึ้นมาเกือบจะทันที มีครบ: วันที่เวลา, เพจ, ชื่อลูกค้า, คำถาม, แพลตฟอร์ม

- [ ] ผ่าน (เช็คใน Sheet ด้วย ไม่ใช่แค่ดูคำตอบในแชท)
